# ДПО МИРЭА

## Описание проекта

Современный лендинг для мебельного магазина Avion — бренда с безвременным дизайном и качественной мебелью. Проект демонстрирует использование современных подходов к вёрстке с применением методологии БЭМ, CSS-переменных и типографических стандартов.

## Используемые технологии

### HTML5
- Семантическая разметка
- Доступность (ARIA-атрибуты)
- Оптимизированная структура документа

### CSS3
- CSS Custom Properties (переменные)
- Flexbox и Grid Layout
- Методология БЭМ (Блок-Элемент-Модификатор)
- Normalize.css для кроссбраузерности
- Hover-эффекты и transitions

### Типографика
- Локально установленные шрифты Roboto (300, 400, 500, 700)
- Типографическая обработка текста (неразрывные пробелы, правильные тире)
- Обработка через типограф для корректного отображения

### Структура проекта
```
DPO-MIREA/
├── assets/
│   ├── favicon.ico
│   ├── fonts/
│   │   └── roboto/          # Локальные шрифты Roboto
│   ├── icons/               # Иконки для UI
│   └── img/                 # Изображения товаров
├── src/
│   └── styles/
│       ├── normalize.css    # Нормализация стилей
│       ├── resets.css       # Сброс стилей
│       ├── fonts.css        # Подключение шрифтов
│       ├── colors.css       # CSS-переменные для цветов
│       └── index.css        # Основные стили
├── index.html               # Главная страница
└── README.md
```
